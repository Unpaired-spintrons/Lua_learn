# 1.1Chunks

是一系列语句，相当于编程当中的一段执行已经完成的指令块

e.g

```lua
a = 1 b = 2*a     --丑陋，但是有效

a = 1;b = 2*a;    --较为规范

a = 1
b = 2*a           --看着十分舒服

a = 1;
b = 2*a;          --C/C++的编程感觉一下就来了
```

其中的**每一行**就是一个**Chunk**这是一个十分有意思的问题，这个主要可以从我们在命令行系统中执行lua的交互模式中可以体现，他代表一个块，一个开始编译的块。

*有趣的是使用一行写很多操作(这样看起来好像很好分块)，所以好几个M的Chunks在lua中常常出现*
